# Ecommerce Admin Dashboard

A SaaS-style ecommerce admin dashboard prototype that demonstrates product management, order processing, customer records, and inventory tracking.

This project simulates a real ecommerce administration panel using **HTML, Tailwind CSS, and JavaScript**, with **LocalStorage acting as a temporary data store**.

---

## Live Demo

https://panchodavao.github.io/commerce-admin-demo-project/login.html

---

## Features

* Product inventory management
* Customer management system
* Order creation and lifecycle tracking
* Cancelled order handling with automatic stock restoration
* Dashboard analytics (products, orders, revenue)
* Low-stock alerts
* Global search across products, customers, and orders
* Order execution date and cancellation date tracking
* Admin login screen with protected dashboard
* Toast notifications for system feedback

---

## Technologies Used

* HTML
* Tailwind CSS
* JavaScript
* LocalStorage (simulated database)

---

## How to Run the Project

1. Clone the repository

https://github.com/PanchoDavao/commerce-admin-demo-project.git

2. Open the project folder

3. Open `login.html` in your browser

No installation or server setup required.
